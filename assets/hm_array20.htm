Example code : Dynamic array instance using New in SmartMS

// Defining an array

var

  a, b : array of Integer;

 

begin

// Instantiate an array

  a := new Integer[5];

  b := a;

  a[1] := 10;

  WriteLn( b[1] );  // 10

 

 

  a.SetLength(10);

  a[1] := 20;

  WriteLn( b[1] );  // 20

Result is: 10 and 20

------------------------------------------------------------------------

 

In other words, in SmartMS, if you want a new dynamic array instance, you use "new", if want to make a copy of a dynamic array, you have to use Copy() method, if you want to resize, you use SetLength() method. See the similar example at Copy() method.

function $DIdxR(a,i,z) {
