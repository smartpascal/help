Now
Function		 Gives the current date and time SysUtils unit
 function Now : TDateTime;
Description
The Now function returns the current date and time in the local time zone.
 
The returned TDateTime type can be manipulated and displayed by many SysUtil and DateUtils routines.
Related commands
Date Gives the current date
Time Gives the current time
Tomorrow Gives the date tomorrow
Yesterday Gives the date yesterday
